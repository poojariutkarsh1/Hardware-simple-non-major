# 🎵 Puranpoli Buzzer Tune (Arduino)

## 📌 Overview
Plays the tune "Puranpoli" using a buzzer connected to an Arduino by generating different frequencies.

## ⚙️ Components
- Arduino  
- Buzzer  

## 🔌 Pin Used
- Buzzer → 9  

## 🚀 Working
- The buzzer plays a sequence of tones using `tone()`
- Delays control note duration and rhythm
- `noTone()` is used to create pauses

## 🎶 Note
Frequencies (like 432, 550, etc.) represent musical notes forming the tune.
